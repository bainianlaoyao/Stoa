**这次实现的结果**
- 新增标准 Vite web 构建，复用现有 renderer 的 Vue/Tailwind/i18n 配置，产物输出到 `stoa-server/dist/web`
- 根脚本新增 `build:web`，并让现有 `build` 先包含 web 构建
- `src/renderer/main.ts` 现在会在 `window.stoa` 不存在时自动走 web bootstrap
- web bootstrap 按 `window.location.origin + ?token=...` 初始化 `StoaClient`，绑定 `window.stoa = StoaClientPreloadAdapter`
- 首次 `getBootstrapState()` 后会自动 `flushBuffer()`，避免 WS 事件永久卡在缓冲区
- Electron 模式保持不变，因为 preload 已经先注入了 `window.stoa`
- `stoa-server --web` 现在实际从 `stoa-server/dist/web` 提供 SPA 静态资源，不再依赖 placeholder
- server 侧静态目录解析做成了多候选路径，兼容从仓库根或 `stoa-server/` 目录启动
- `index.html` 的 CSP 放开了 `http/https/ws/wss` 连接，避免 web 模式下 WS 被 CSP 拦截
